# Gravitational Wave Prediction of Supermassive Blackholes In Cosmological Simulations


Analysis of supermassive black hole (SMBH) mergers from the IllustrisTNG **TNG-300** simulation and their detectability by **LISA** (Laser Interferometer Space Antenna).

## Contents

- `Supermassive_Blackhole_mergers_TNG_300.ipynb` — main analysis: loads the merger catalog, computes GW strain/SNR, and evaluates LISA detectability.
- `Mass_Distribution.ipynb` — SMBH mass distribution analysis.
- `blackhole_mergers_tng300.hdf5` — TNG-300 BH merger catalog.
- `LISAstrain.txt` — LISA sensitivity curve (frequency vs. characteristic strain).
- `Age.npy` — age of the universe (Gyr) at each merger redshift.
- `Mass Distribution.xlsx` — supporting mass distribution data.
- `GW_Predictions_Poster.pptx` — poster summarizing findings.

## Usage

Open the notebooks in Jupyter and run cells top to bottom; each notebook documents its own data-loading and analysis steps inline.
